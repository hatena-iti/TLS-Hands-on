# TLS-Hands-on
TLS1.2のプロトコルを簡易にし、おおよその流れが分かるようにしたハンズオン資料です。

## 注意
- 厳密なTLS1.2のプロトコルではありません
- 間違いが含まれる可能性があります
- 本資料を使った通信における安全性は保障しません
- 本資料のライセンスはCC-BY-NC-SA 4.0(表示 - 非営利 - 継承)です

## 使い方

### ハンズオン用チャットの開設

DiscordやSlackなど任意のチャットツールで本ハンズオン用のチャットを開設してください。
例：#TLSハンズオンチャンネル

### 役割分担

次に進行役、サーバー役、クライアント役を決めます。

| 役 | 役割 | 参照すべきファイル |
| - | - | - |
| 進行役（TLSについてざっくりと理解している人間が望ましい） | 全体進行とCA役としての事前準備 | [TLS体験_ca.ipynb](TLS体験_ca.ipynb) |
| サーバー役 | TLS通信のおけるサーバーとしてふるまう | [TLS体験_server.ipynb](TLS体験_server.ipynb) |
| クライアント役 | TLS通信のおけるクライアントとしてふるまう | [TLS体験_client.ipynb](TLS体験_client.ipynb) |
| その他 | パケットを盗聴し、サーバとクライアントの通信を改ざん、盗聴を試みてもいい | - |
